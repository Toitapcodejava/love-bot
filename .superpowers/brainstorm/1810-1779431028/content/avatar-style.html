A — Emoji Reactive

Avatar hình tròn gradient, có badge emoji nhỏ ở góc thể hiện cảm xúc hiện tại. AI phân tích mood → chọn emoji phù hợp (😤 khi user tức, 🥺 khi user buồn, 😏 khi Kem sắp châm biếm...). Đơn giản nhưng hiệu quả.

[ IDLE ] [ TALK ] [ THINK ] [ MAD ]
pixel sprite với 4 animation state

B — Pixel Sprite

Nhân vật pixel art nhỏ (~64px) với 4 trạng thái animation: idle (thở nhẹ), talk (miệng động), think (dấu "..."), mad (rung). Bắt mắt, retro-cute, tốn công làm nhất nhưng rất độc đáo.

C — Smooth SVG Face

Mặt SVG vector đơn giản, animate mượt (không cần asset file). Mắt, miệng, lông mày đều animate được. Nhiều biểu cảm hơn emoji, nhẹ hơn Lottie/pixel art, có thể làm hoàn toàn trong code.

Click để chọn, hoặc nói trong terminal nếu muốn kết hợp.
