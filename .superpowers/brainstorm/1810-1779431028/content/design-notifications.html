Cơ chế kỹ thuật

  • Backend endpoint mới — POST /notify/schedule lên lịch gửi, POST /notify/send trigger ngay
  • Context cho Claude — giờ hiện tại + last_chat_hours_ago + last_mood từ memory
  • App đã có lib/notifications.ts — chỉ cần thêm handler nhận push từ backend
  • Expo Push Token — lưu token khi app khởi động, gửi lên backend 1 lần
  • Rate limit — tối đa 3 notification/ngày, không gửi 11pm–8am (trừ khi user đang online)
