**Goal:** Add a GPS tab to the Expo app that uses foreground location (works in Expo Go) to send location data to the backend and display current position + AI suggestions.

**Architecture:** A new tab `app/(tabs)/gps.tsx` calls `Location.getCurrentPositionAsync()` (foreground only, works in Expo Go), POSTs to `/location/update`, then GETs `/location/suggest`. The existing background task in `app/lib/location.ts` is untouched. No backend changes needed.

**Tech Stack:** expo-location (already installed), react-native-maps (bundled in Expo SDK), expo-router tabs, TypeScript

- [ ] **Step 2: Add `fetchLocation` function that requests permission and gets position**

Add this function inside the `GPS` component body, before the `return`:

```typescript
async function fetchLocation() {
  setState("loading");
  setErrorMsg("");

  const { status } = await Location.requestForegroundPermissionsAsync();
  if (status !== "granted") {
    setState("error");
    setErrorMsg("Bạn của Kem cần quyền truy cập vị trí.");
    return;
  }

  let pos: Location.LocationObject;
  try {
    pos = await Location.getCurrentPositionAsync({
      accuracy: Location.Accuracy.Balanced,
      timeInterval: 15000,
    });
  } catch {
    setState("error");
    setErrorMsg("Không lấy được vị trí. Thử lại?");
    return;
  }

  const { latitude: lat, longitude: lng, accuracy } = pos.coords;

  // Reverse geocode
  let address = `${lat.toFixed(4)}° N, ${lng.toFixed(4)}° E`;
  try {
    const geo = await Location.reverseGeocodeAsync({ latitude: lat, longitude: lng });
    if (geo.length > 0) {
      const g = geo[0];
      const parts = [g.street, g.district, g.city].filter(Boolean);
      if (parts.length > 0) address = parts.join(", ");
    }
  } catch {
    // keep default address
  }

  // POST to backend
  const base = await storage.getBase();
  const key = await storage.getKey();
  try {
    await fetch(`${base}/location/update`, {
      method: "POST",
      headers: { "x-app-key": key, "content-type": "application/json" },
      body: JSON.stringify({ lat, lng, accuracy }),
    });
  } catch {
    // silently ignore — location still usable locally
  }

  setLocation({ lat, lng, accuracy: accuracy ?? null, address, updatedAt: new Date() });
  setState("ready");
}
```

Add this function before `export default function GPS()`:

```typescript
function formatRelativeTime(date: Date): string {
  const diff = Math.floor((Date.now() - date.getTime()) / 1000);
  if (diff < 60) return "vừa xong";
  if (diff < 3600) return `${Math.floor(diff / 60)} phút trước`;
  return `${Math.floor(diff / 3600)} giờ trước`;
}
```
